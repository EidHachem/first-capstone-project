Main Program

Roadmap

Anyone who wants to learn something must have a clear path to follow in order to achieave his goal, here we will tell you what we think is the best way to learn web development

HTML

A short briefing about the language, what are the the most important tags, and why you should learn HTML5.

CSS

A short briefing about the language, what are the best practices, and why you might want to learn css preprocessors or framworks.

Javascript

JS is the most important skill for a web developer, we will give you the most important reasons to master Js, and the wat to achieve it.

React

Js frameworks are so important those days, and React is the most popular one nowadays, we will talk a bit about the importance of a framework and how it can help you while building interactive websites.
